- 5분 지각 → 5천원

## 문서

- 깃헙 위키 페이지(OT 이후에 카톡 채팅방에 공유할 예정)

## 스터디 방식

- OT에서 책의 분량을 확인하고 몇명이 어느 범위를 업로드 할지 정한다. (ex. 1주에 세명이 아이템 4개? 5개?)
- 매 주 마다 사전에 정한 분량을 위키 페이지에 업로드 한다. ( 최대한 발표 전날까지는 업로드 완료 )
- 오프라인 모임에서는 위키 페이지에 업로드 한 인원들끼리 3등분 하여 발표를 진행한다. (대략 1인당 아이템 1-2개?)
- 오프라인에서 새롭게 올라온 주제는 위키페이지에 공유하고 지원자를 선정하여 추가적인 정보를 모임에 공유해준다. (위키 페이지 댓글로 공유)
